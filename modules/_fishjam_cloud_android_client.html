Module @fishjam-cloud/android-client

Fishjam Android Client

Android client library for Fishjam.

[!WARNING]
This SDK is not stable yet. We recommend to use React Native Client for Fishjam Cloud services.

Installation

Add jitpack repo to your build.gradle:

 allprojects {
  repositories {
   ...
   maven { url 'https://jitpack.io' }
  }
 }

Add the dependency:

 dependencies {
   implementation 'com.github.fishjam-cloud:android-client-sdk:<>'
 }

Usage

Make sure you have:

  • Running Fishjam Cloud account.
  • Created room and token of peer in that room. You can use Room Manager from your account dashboard

Development

  1. Set FISHJAM_SOCKET_URL in ~/.gradle/gradle.properties to your dev backend.
  2. Run ktlint to format code (if missing, install it with brew install ktlint)
