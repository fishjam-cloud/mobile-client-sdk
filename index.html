Fishjam Cloud Mobile Client

Monorepo containing mobile SDKs for Fishjam server

  • android-client
  • ios-client

Documentation

API documentation is available here: Android iOS React Native

Example

We strongly recommend checking out our example app that implements a basic video room client. To run the app:

  1. Go to Membrane's server demo repo: https://fishjam.io/app/login Follow instructions there to setup and run demo server.
  2. Clone the repo
  3.    yarn
       yarn build
       cd `examples/fishjam-chat`
       npx expo prebuild --clean
       yarn run android/ios

[!NOTE] Be sure to run npx expo prebuild --clean and not yarn prebuild --clean as there's an issue with path generation for ios/.xcode.env.local

  1. Follow further instructions from README

Developing

Run ./scripts/init.sh in the main directory to install swift-format and set up git hooks.

Follow further instructions for the SDK you're developing for.

Releasing

Follow instructions from RELEASE.md The code should be released using our release.yml Github Action.

This script will:

  1. Fetch the dependencies
  2. Run expo-module prepare to clean, configure and build source code, plugins etc. See for more details
  3. Publish to npm.
